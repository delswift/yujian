## 为什么是御剪

很多理发店老板不缺“软件”，缺的是一套**真正能落地、能收钱、能管人、能沉淀会员**的系统。

御剪不是做给“大而全 PPT”看的，而是针对理发店真实经营场景打造：

- **日常经营本地单机运行**
- **会员、员工、收银、报表本地即可使用**
- **只有订阅续费 / 授权验证时才连接官方云计费服务**
- **店长 / 店员角色分明，权限可控**
- **不仅能收银，更能做会员沉淀、提成核算和连锁经营分析**

一句话概括：

> **先帮门店把生意跑起来，再帮老板把经营管起来。**

---

## 产品定位

御剪当前采用两段式产品结构：

### 1. 门店端：Windows 单机版
- 单文件运行
- 数据本地保存
- 断网也不影响大部分日常经营
- 适合单店、夫妻店、工作室、连锁门店前台主机

### 2. 官方云计费服务
- 只负责订阅收费、续费、退款、补单、对账
- 统一托管微信 / 支付宝真实收款参数
- 门店端不暴露开发者收款配置

这意味着：

- 门店平时经营不依赖公网
- 你的收费链路又可以统一掌控
- 安全性和落地性都更适合当前市场

## 快速开始

### 方式一：直接使用发布包
如果你已经拿到打包好的发布文件，直接运行：

```text
御剪单机版.exe
```

首次打开后：
1. 进入安装向导
2. 填写门店名称、管理员账号和密码
3. 完成初始化后即可开始使用

> 注意：系统没有默认账号和默认密码，首次安装时自行创建。
